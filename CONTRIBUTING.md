# Contributing to bracu.network

Thank you for your interest in contributing!

## Code Contributions
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Ensure the linter and build pass locally (`npm run lint && npm run build`)
5. Submit a Pull Request

## Adding Yourself via PR (Alternative to Web Form)
If you prefer adding yourself directly via code instead of the `/join` form:

Currently, the primary way to join is via the web form which inserts into the Supabase backend.
If you'd like to submit a PR to modify the database schema or core code, follow the standard code contribution guidelines.

Please do not submit PRs with raw SQL inserts for yourself unless specifically discussed with the maintainers. Use the `/join` form instead!
